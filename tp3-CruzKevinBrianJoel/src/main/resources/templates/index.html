¿Que es Java?

Java es un lenguaje de programacion y una plataforma Informatica que fue comercializada por primera vez en 1995 por Sun Microsystems.

Creacion

Java fue creado en 1991 por un equipo de ingenieros liderado por James Gosling en Sun Microsystems. La primera versión pública, Java 1.0, se lanzó en 1995. Desde entonces, Java se ha convertido en uno de los lenguajes de programación más populares y utilizados.

  • Su enfoque en la portabilidad y capacidad para ejecutarse en diferentes plataformas sin necesidad de reescribir el código lo ha hecho muy popular.
  • Se utiliza ampliamente en aplicaciones de escritorio, web, móviles y empresariales.
  • Java cuenta con una amplia comunidad de desarrolladores y un ecosistema de herramientas y frameworks que facilitan el desarrollo de software.

Filosofia

El lenguaje Java se creó con cinco objetivos principales:

  • Debería usar el paradigma de la programación orientada a objetos.
  • Debería permitir la ejecución de un mismo programa en múltiples sistemas operativos.
  • Debería incluir por defecto soporte para trabajo en red.
  • Debería diseñarse para ejecutar código en sistemas remotos de forma segura.
  • Debería ser fácil de usar y tomar lo mejor de otros lenguajes orientados a objetos, como C++.
